## 使用说明

> 1. 按照程序指引即可，可辅助键盘输入，不推荐再已存在的计算结果的同一行键入，不美观，但不影响程序运行；
>
> 2. 程序所能显示数据大小为18位数，超过此位数将自动进行四舍五入科学计算，但只会在最终结果处近似处理，因而不会影响结果精度，请放心使用；
> 3. 文本附带保存文件，请不要随意删除。
